World Record for Android app is released

The World Record app is released on Android Play store.

  • World Record

Development Background

Couple of months ago, I am looking for the World Record data on the internet but I couldn’t find what I want easily. I checked the wikis and each sports websites but It is very hard to find the World Record data. So I decided to make World Record app, So that people can find the World Record data very easily.

App Development

I made this app using Flutter. Because, using Flutter I can make Android app and iOS based on one programming language, Dart. I had used React Native for a six months before I knew Flutter but there are a lot of problem with React Native. React Native‘s performance was not good and there are a lot of errors displayed when I build application. In addition to that, I couldn’t use Admob. Yes, there is Admob plugins but It seems that the Admob plugins are not maintained. That’s why I developed the World Record app using Flutter.

App Description

Using this App, you can now find the World Record data very easily. Currently, there are 7 sports, Athletics, Inline Speed Skating, Short Track, Ski Jumping, Speed Skating, Swimming and Track Cycling. Once you click one of the sports, there are many list for the sport. You can find the World Record data and who holds the World Record data and also you can check when and where the words record is broken.

Maintenance

Once there is a new World Record, then I’ll update this app as soon as possible. If there is wrong information, then please let me know, so that I can update this app.

Upcomming

I’ll add more sport categories.
Also, there will be an iOS app for iPhone users.
